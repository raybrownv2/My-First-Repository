# 決算・申告 チェックリスト

税理士に指摘された決算処理・税務申告に関するチェック項目をまとめています。

---

## 定期チェック項目

### 期末決算
- [ ] 売掛金・買掛金の残高確認と照合が完了しているか
- [ ] 棚卸資産の実地棚卸を実施し、帳簿と一致しているか
- [ ] 減価償却費が正確に計上されているか
- [ ] 未払費用・前払費用の計上漏れがないか
- [ ] 貸倒引当金の設定が適切か
- [ ] 役員賞与・役員報酬の変更手続きが適切に行われているか

### 税務申告
- [ ] 法人税・消費税の申告期限を確認しているか
- [ ] 中間申告・予定納税の漏れがないか
- [ ] 交際費の損金算入限度額を超えていないか（800万円基準）
- [ ] 繰越欠損金の控除が適切に適用されているか
- [ ] 税務上の引当金・準備金の要件を満たしているか

### 消費税
- [ ] 課税売上割合の計算が正確か
- [ ] 非課税・免税・不課税取引の区分が適切か
- [ ] 簡易課税制度の適用可否を確認しているか
