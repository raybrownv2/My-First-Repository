# 税理士指摘ナレッジベース

税理士に指摘されたチェック項目を蓄積・共有するためのナレッジベースです。

## ディレクトリ構成

| ファイル | カテゴリ |
|---|---|
| [経費・仕訳](./expenses-journal.md) | 経費処理・仕訳に関するチェック項目 |
| [インボイス・請求書](./invoice-billing.md) | インボイス制度対応・請求書の記載事項 |
| [決算・申告](./settlement-filing.md) | 決算処理・税務申告に関するチェック項目 |
| [給与・源泉徴収](./payroll-withholding.md) | 給与計算・源泉徴収に関するチェック項目 |

## チェック項目の記載フォーマット

各ファイルでは以下のフォーマットで項目を追記してください。

```markdown
### [YYYY-MM] 指摘内容の概要

- **指摘内容**: 税理士から指摘された具体的な内容
- **原因**: なぜ発生したか
- **対応方法**: どのように修正・対応したか
- **再発防止策**: 今後どう防ぐか
```

## 更新ルール

- 新しい指摘を受けたら、該当カテゴリのファイルに追記する
- 指摘を受けた年月を見出しに記載する
- 既存項目と重複する場合は、備考として追記する
