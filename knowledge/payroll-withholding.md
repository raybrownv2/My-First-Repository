# 給与・源泉徴収 チェックリスト

税理士に指摘された給与計算・源泉徴収に関するチェック項目をまとめています。

---

## 定期チェック項目

### 給与計算
- [ ] 給与規定に基づいた計算が行われているか
- [ ] 社会保険料（健康保険・厚生年金）の控除額が正しいか
- [ ] 雇用保険料の控除額が正しいか
- [ ] 時間外労働・休日労働の割増賃金が正確に計算されているか
- [ ] 通勤手当の非課税限度額（月15万円）を超えていないか

### 源泉徴収
- [ ] 源泉所得税の徴収額が正しいか（税額表の適用確認）
- [ ] 扶養控除等申告書を従業員から受領しているか
- [ ] 特別徴収住民税の金額が市区町村の通知と一致しているか
- [ ] 源泉所得税の納付期限を守っているか（翌月10日、納期特例は1月・7月）

### 年末調整
- [ ] 年末調整の対象者・非対象者を正しく区分しているか
- [ ] 各種控除申告書（保険料・住宅ローン等）を回収しているか
- [ ] 過不足税額の精算が12月（または1月）の給与に反映されているか
- [ ] 法定調書（源泉徴収票、給与支払報告書）を期限内に提出しているか
- [ ] 年末調整後の納付額に誤りがないか

### 役員報酬
- [ ] 役員報酬の定期同額給与要件を満たしているか
- [ ] 事前確定届出給与の届出が期限内に行われているか
