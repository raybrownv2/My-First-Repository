# 経費・仕訳 チェックリスト

税理士に指摘された経費処理・仕訳に関するチェック項目をまとめています。

---

## 定期チェック項目

- [ ] 領収書・レシートの宛名が法人名になっているか
- [ ] 10万円以上の備品は固定資産として計上しているか（少額減価償却資産の特例適用確認）
- [ ] 交際費と会議費の区分が適切か（5,000円基準の確認）
- [ ] 経費の勘定科目が正しく分類されているか
- [ ] 個人的な費用が混入していないか
- [ ] 現金払いの経費に領収書が揃っているか
