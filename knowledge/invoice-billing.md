# インボイス・請求書 チェックリスト

税理士に指摘されたインボイス制度対応・請求書に関するチェック項目をまとめています。

---

## 定期チェック項目

### 適格請求書（インボイス）の記載事項
- [ ] 適格請求書発行事業者の登録番号（T+13桁）が記載されているか
- [ ] 取引年月日が記載されているか
- [ ] 取引内容（軽減税率対象の場合はその旨）が記載されているか
- [ ] 税率ごとに区分した合計額（税抜または税込）が記載されているか
- [ ] 税率ごとに区分した消費税額が記載されているか
- [ ] 書類の交付を受ける事業者の氏名または名称が記載されているか

### 受領した請求書の確認
- [ ] 仕入先がインボイス登録事業者であるか確認済みか
- [ ] 未登録事業者からの仕入れに対する経過措置（仕入税額控除の割合）を適用しているか
- [ ] 請求書の保存期間（7年間）を守っているか

### 発行した請求書の確認
- [ ] 自社の適格請求書発行事業者登録番号を記載しているか
- [ ] 消費税の内訳（8%・10%）が明記されているか
